# FireFly

## Installation


```bash
git clone https://github.com/DamodarAgrawal/Firefly.git
cd FireFly
```
Backend setup
```bash
cd firefly-backend
npm install
npm start
```
Frontend setup
```bash
cd firefly-frontend
npm install
npm run dev
```
Admin Frontend setup
```bash
cd firefly-admin-frontend
npm install
npm run dev
```
